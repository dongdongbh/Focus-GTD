# Mindwtr 0.5.8

## Highlights
- Android capture now defaults to text input, with clearer inbox hints when voice capture is enabled.
- Desktop inbox processing no longer repeats the Project step when “Project first” is enabled.
- Board view can expand to full width on large screens.

## Desktop
- Inbox processing respects “Project first” without asking for Project twice.
- Board view uses full width to show more columns on wide monitors.

## Mobile
- Default capture method is text for new installs.
- “Check for updates” opens the Google Play listing on Android.
- Reference view no longer shows quick-add.
- Projects page removes duplicate area filter row.

## Full Change List (since 0.5.7)
- fix(desktop): avoid duplicate project step
- fix(desktop): allow full-width board view
- fix(mobile): default to text capture
- fix(mobile): open play store for update checks
- fix(mobile): hide quick add in reference view
- fix(mobile): avoid duplicate area filters
- docs: add google play install links
