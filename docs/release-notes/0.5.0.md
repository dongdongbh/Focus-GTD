# Mindwtr v0.5.0

## Highlights
- New theme picker on mobile with Material 3 Light/Dark plus E‑Ink, Nord, Sepia, and OLED (Midnight).
- Desktop now includes E‑Ink, Nord, and Sepia themes.
- Mobile headers and navigation bars now follow the active theme.
- Project list actions on mobile match desktop (trash icon + reorder on the right).

## Security
- AI API keys now use secure storage on mobile (SecureStore) and native config on desktop.
